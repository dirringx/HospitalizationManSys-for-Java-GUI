# HospitalizationManSys-for-Java-GUI
基于JavaGUI实现的医院住院管理系统
IED为Eclipse  
MySQL数据库，数据库名：hosmansys  
数据库登录名为root，密码为admin  
起始页面：LogInUI.java  
工作区（workspace）编码集：GBK
